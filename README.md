## 🚀 Overview  
Responsive e-learning platform focused on usability and progress tracking.

## 🛠 Tech Stack  
HTML5 • CSS3 • JavaScript • Tailwind

## 📦 Local Setup  
1. Clone repo  
2. `npm install`  
3. `npm start`

- User progress tracking  
- Modular UI  
- Designed for maintainability
