# Meu Jarvis – AI Web Application

**Live Demo:** https://meu-jarvis-inky.vercel.app

## 🚀 Project Overview
Full-stack web application built with React.js, Next.js and TypeScript.

- Front-end modular UI with React + Next  
- Authentication with Supabase (PostgreSQL)  
- Dynamic AI responses powered by Groq API  
- Deployed on Vercel with secure environment configs

## 🛠 Tech Stack
React.js • Next.js • TypeScript • Supabase • PostgreSQL • REST API • Vercel

## 📦 Run Locally
1. `git clone https://github.com/LuisGustavoCassioli/meu-jarvis`  
2. `npm install`  
3. Create `.env.local` with Supabase env vars  
4. `npm run dev`

## 📌 Why This Project Matters
Shows ability to build production-grade apps with auth, API integration, database persistence, and real deployment.
